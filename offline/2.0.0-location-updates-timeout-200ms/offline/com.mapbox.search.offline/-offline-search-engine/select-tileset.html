selectTileset

abstract fun selectTileset(dataset: String?, version: String?)

Selects preferable tileset for offline search. If dataset or version is set, OfflineSearchEngine will try to match appropriate tileset and use it. If several tilesets are available, the latest registered will be used.

By default, if multiple tilesets are registered at once (e.g. one offline region is loaded for several tilesets in single call), the biggest one (tilesets are compared as pair of strings {dataset, version}) is considered as latest.

Parameters

dataset

Preferable dataset.

version

Preferable version.
